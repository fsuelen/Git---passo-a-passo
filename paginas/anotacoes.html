// Comandos Git //


  1. * Git init = inicializar Git
  2. * Git rm -rf + nome do arquivo = tirar do git
    (perde todos os commits caso exista algum)
  3. * Git status = ver como anda o status do projeto
  4. * Git commit -am para arquivos que já foram comitados antes. pega o commit e ja add na estage e commita logo em seguida
  5. * Git log = mostra todos os commits até o momento
  6. * Git log -1 -2 -3 = mostra o ultimo ou os dois ultimos e assim sucessivamente
  7. * Git log oneline = commits reduzidos para caber em apenas 1 linha
  8. * Git passando uma rash show = mostra tudo que foi feito naquele commit
  9. * Git log -1 --stat = Diz quantas linhas foram alteradas
  10. * Git log --autor = + nome me diz quem foi o nome do autor daquele commit
  11. * Git log --parent --online = mostra a arvore de commits pai e filhos
  12. * Git diff = mostra exatamente o que eu alterei antes de dar o commit
  13. * Git diff --cached mostra o que eu alterei estando o arquivo em stage
  14. * Git diff rash .. rash mostra o que alterei entre 1 commit e outro
  15. * Git reset -- soft + rash desfaz o commit e volta pra stage
  16. * Git seret -- mixed + rash desfaz o commit e tira da stage
  17. * Git reset --hard + rash desfaz todo o commit e apaga todas as alterações feitas
  18. * Git revert + rash do commit que quero reverter e coloco --no-edit = reseta e mantem o texto do commit removido
  19. * Git mv nome do arquivo atual + nome do arquivo modificado = altera nome do arquivo e já deixa em stage
  20. * Git add remote + nome do repositorio remoto + link do repositorio remoto = add um repositorio remoto
  21. * Git remote -v = mostra o link do repositorio remoto
  22. * Git push + nome do reositorio remoto + master + push = push que é = a enviar para nuvem
  23. * Git clone + link do repositorio remoto + nome que tu quiser = clona o projeto como estava na nuvem
  24. * Git pull + nome do repositorio + master = pegar o que foi alterado no remoto por outras pessoas
  25. * Git Branch = lista as branch que eu tenho
  26. * Git Branch + nome da branch = adicionar nova branch
  27. * Git checkout + nome da branch = trocar de branch
  28. * Git checkout -b + nome = adiciona nova branch e vai pra ela na mesma linda de comando
  29. * Git --log --all = mostra todos os comites, mesmo estando em branch diferentes
  30. * Git merge + nome da branck que quer mesclar = mescla as branch adicionando um novo commit porém segue o histórico dos commits
  31. * Git Branch -d + nome da branch = exclui ela, poré n pode estar nela nesse momento
  32. * Git log -- grafh = mostra o grafico pra mim das minhas branchs e commits feitos
  33. * Git rebase + nome da branch = mescla também com a master, mas sem manter o histórico mas mantem o gráfico mais limpo
  34. * Git branch -D = exclui os commits e as branch junto
  35. * Git branch -r = mostra branch remota
  36. * Git branch -a = motra todas as branches, local e remota
  37. * Git push origin + nome da branch = da um push da nova branch e cria uma igual remota
  38. * Git push -u + nome da branch traqueia com a remota
  39. * Git push + nome do repositorio remoto + : nome da branch remota exclui a branch remota
  40. * Git congig -l = mostra toda a lista de abreviaçõs dos comandos
  41. * Git config alias . mas nome criado do alias + entre "" colocar nome do comando que tu quer
  42. * Git config --unset alias + nome que tu configuou do alias que tu critou = deleta alias gi
