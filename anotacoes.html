* Git init = inicializar Git
* Git rm -rf + nome do arquivo = tirar do git
(perde todos os commits caso exista alum)
* Git status = ver como anda o status do projeto
* Git commit -am para arquivos que já foram comitados antes. pega o commit e ja add na estage e commita logo em seguida
* Git log = mostra todos os commits até o momento
