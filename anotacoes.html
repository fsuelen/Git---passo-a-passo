// Anotações //


* Git init = inicializar Git
* Git rm -rf + nome do arquivo = tirar do git
(perde todos os commits caso exista alum)
* Git status = ver como anda o status do projeto
* Git commit -am para arquivos que já foram comitados antes. pega o commit e ja add na estage e commita logo em seguida
* Git log = mostra todos os commits até o momento
* Git log -1 -2 -3 = mostra o ultimo ou os dois ultimos e assim sucessivamente
* Git log oneline = commits reduzidos para caber em apenas 1 linha
* Git passando uma rash show mostra tudo que foi feito naquele commit
* Git log -1 --stat = Diz quantas linhas foram alteradas
*Git log --autor = + nome me diz quem foi o nome do autor daquele commit
